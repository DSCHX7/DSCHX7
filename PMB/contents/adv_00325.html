  • 操作系统：Microsoft Windows XP SP3*/Windows Vista SP2**/Windows7
    *不支持64位版本和Starter（Edition）。
    **不支持Starter（Edition）。
    需要执行标准安装。

  • CPU：依您的照相机型号而异。请参阅照相机附带的使用说明书。
  • 应用程序：DirectX 9.0c或更新版本（本产品基于DirectX技术。需要安装DirectX。）
    Windows Media Player 9或更新版本。
  • 内存：依您的照相机型号而异。请参阅照相机附带的使用说明书。
  • 显示：最小1024 x 768像素

注意
